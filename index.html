About Strum

Strum allows you to learn how to play guitar with a Sensel. The Sensel Morph is a "pressure-sensitive, multi-touch input device that enables users to interact with the digital world". It has a lot of applications, from multi-touch games to modeling based education. This project is inspired by one of the pre-existing overlays - the Piano overlay - and aims to convert the Morph into a standard 6-string guitar.
